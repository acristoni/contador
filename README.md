## Desafio de código inicial do bootcamp DIO fundamentos de Javascript

Desafio proposto pela professora Stephany Nusch no curso de introdução ao Javascript da Digital Innovation One Inc.

Tomei a liberdade de acrescentar mais uma coisinha no desafio, que foi a impressão no console da mensagem para a impossibilidade de usar o botão de acréscimo quando o valor for maior que 10.

Consegui também unir os dois desafios em uma única solução deixando os números vermelhos quando forem menores que zero e "travando" o botão quando maiores que 10.
